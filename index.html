Haptic Neural Fields: Bringing Tactile Interactions to 3D Rendered Scenes

Antonio Luigi Stefani, Niccolò Bisagno, Nicola Conci, Eckehard Steinbach, Francesco De Natale

Our demo

>

What would objects feel like if touched in a virtual environment?

How does an object feel when stroked with a hand? We propose a method that augments 3D scenes with object-specific tactile feedback. We train a neural field to model haptic responses conditioned on a surface material and an action. By integrating material properties with the underlying 3D geometry, the learned fields enable interactive tactile exploration: at test time, a user can input an arbitrary hand action, and the model estimates the resulting tactile signal. Here we show examples of hand interactions and the corresponding generated haptic feedback.

Abstract

We address the problem of making 3D scenes interactive by asking: what would objects feel like if touched in a virtual environment? State-of-the-art 3D rendering methods provide compelling visual realism, but they fall short in modeling physical interactions, such as haptic feedback. We propose a framework that learns the correspondence between user actions and tactile responses, enabling the generation of touch-based signals directly from simulated interactions in 3D scenes. Our approach leverages a neural field representation conditioned on geometry and action to synthesize material-specific tactile signals. Experiments show that the generated signals reliably convey material properties and interaction dynamics. This paves the way toward interactive, touch-aware virtual environments with realistic haptic feedback.

Pipeline overview. From a 3D scene (NeRF/3DGS) we render an RGB view, use a conditional diffusion model to obtain a co-registered haptic map I, and encode it into a material token m = E(I). An action description is encoded as u(t). The Haptic Neural Fields (HNF) consumes (m, u(t)) and outputs a vibrotactile acceleration signal a(t).

Overview of the Haptic Neural Field (HNF) and temporal rendering. (a) Given a contact trajectory on a material surface, we sample directions, speeds, and normal forces over time. (b) For each sample, an MLP conditioned on the haptic map and actions outputs a local emitted acceleration contribution and a nonnegative haptic density. (c) A transmittance-based temporal renderer composites these values over a short causal window to produce the vibrotactile acceleration signal. (d) The HNF is trained end to end with a frequency-domain loss that minimizes the residual between the synthesized and ground-truth DFT magnitudes.

Do directions influence haptic stimuli generation?
With Haptic Neural Fields they do

Direction-dependent haptics. Left: two scratch trajectories on the same material (purple: top→down, strong force; green: left→right, strong force). Right: acceleration DFT magnitude of the synthesized signals for each direction (color frames match the trajectories). HNF produces distinct, direction-specific spectra—redistributing energy and revealing anisotropy—whereas the GACM baseline yields similar, low-pass like spectra for both directions, failing to model directional effects.

Some qualitative results

Qualitative, action-conditioned touch in 3D scenes. From a rendered scene (NeRF/3DGS, left), we select contact locations (color-coded insets), render an RGB crop, and generate a co-registered haptic map. Each map is material-classified to route conditioning. Given an action description (e.g., scratch / rub with soft-hard or slow-fast profiles), HNF synthesizes the corresponding vibrotactile acceleration trace (right), producing distinct signals for different materials and actions at the same site.
